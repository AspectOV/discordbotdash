Privacy Policy

Last Updated: 2/4/2026

This Privacy Policy describes how A.S.P.E.C.T (the “Bot”) collects, uses, and shares information when you use the Bot in Discord servers.

What We Collect

  • Identifiers: Discord User IDs, Server (Guild) IDs, Channel IDs.
  • Bot interaction data: Data required to provide features (e.g., economy balances, settings).
  • AI chat data (Chat.py): Message content submitted for chatbot functionality may be stored in a database controlled by the Bot operator.

How We Use Data

  • Provide Bot features and server configuration.
  • Operate and improve AI chat functionality.
  • Security, abuse prevention, and debugging.

Third-Party Processing (Google Gemini)

When you use AI chat features, message content may be sent to a third-party AI provider (Google Gemini) to generate responses. This processing occurs only for AI chat functionality.

Retention

We retain data only as long as necessary to provide features, maintain security, and operate the Bot. Retention periods may vary depending on the feature (e.g., economy data vs. chat logs).

Data Deletion Requests

You can request deletion of stored data by contacting: support@aspectbot.app. Include your Discord User ID and the server where you used the Bot.

Security

We use reasonable safeguards to protect data stored in systems we control. However, no system can be guaranteed 100% secure.

Changes

We may update this policy from time to time. Continued use of the Bot after changes means you accept the updated policy.
