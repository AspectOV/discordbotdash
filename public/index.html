Core Features

AI Chatbot

Slash-command chatbot that can use Google Gemini to generate responses. Designed to be optional and invoked intentionally, not always-on message scraping.

Economy

Balances, rewards, and progression with room to expand into server shops, perks, and configurable earn rates.

Roles & Automation

Tools for roles and automation workflows. Built to reduce repetitive admin tasks and keep servers organized.

Welcome & Utility

Welcome messages, setup helpers, and practical utilities that make server management smoother.

Engagement

Features that reward active members and support community momentum (boosters, fun tools, and more).

Dashboard-ready

The architecture is built to grow into a full configuration dashboard and docs under aspectbot.app.

How it works

1) Invite

Add A.S.P.E.C.T using the invite link and choose permissions intentionally.

2) Configure

Use slash commands today, and the dashboard for settings as it rolls out.

3) Run

Let the bot handle routine server tasks while you focus on your community.

Note on AI chat: AI responses may be inaccurate. Use it for help and fun, but don’t treat outputs as professional advice. Full details are in the Privacy Policy and Terms.
